⚙️ ONE-TIME FIREBASE SETUP — connects all devices to the same database

  1. Go to console.firebase.google.com → Create a project (disable Analytics, not needed)
  2. In your project: Firestore Database → Create database → Start in test mode → pick any location → Enable
  3. Click ⚙️ Project settings → Your apps → </> Web icon → register app → copy the firebaseConfig object
  4. In this app: Admin (password: admin123) → Firebase Config tab → paste your config → Save & Connect
  5. Once connected on your device, everyone who opens the link will share the same data automatically
